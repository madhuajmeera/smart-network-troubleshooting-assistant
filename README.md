# Smart Network Troubleshooting Assistant

A telecom support automation platform developed using Python, Flask, SQLite, HTML, CSS, and Bootstrap.

## Features

* Automated troubleshooting for network issues
* Mobile data diagnostics
* SMS and call issue resolution
* Ticket generation and tracking
* Admin dashboard
* Network Health Analyzer
* AI-assisted troubleshooting recommendations

## Technologies Used

* Python
* Flask
* SQLite
* Bootstrap
* HTML
* CSS
